What is Front End Development?

Front-end web development is the development of the graphical user interface of a website, through the use of HTML, CSS, and JavaScript, so that users can view and interact with that website. Front End recieves the Data from Back End through APIs.

  • Media Query is a CSS Query which is used for handling responsiveness under certain situations given to the Query

    • Media Query in CSS

    Grid

  • Grid is used to give a specific area to a specific Components

    • We usually use grid in Bootstrap as col-span-md-3

    We adjust the Specific size allocated to the component using properties such as:

  • xs (Extra Small) Device
  • sm (Small) Device Mobile || Phones etc
  • md (Medium) Device || Tabs Etc
  • lg (Large) Device || PCs and Laptops

    • Flexbox

    Flexbox is hands down the best way to give position to the element.

    FLEXBOX
  • The Flexible Box Module, usually referred to as flexbox, was designed as a one-dimensional layout model, and as a method that could offer space distribution between items in an interface and powerful alignment capabilities.
  • Flexbox became much popular after Responsiveness became more popular and Webpage's positioning became much more important then earlier.
